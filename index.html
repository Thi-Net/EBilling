Fitur inti yang benar-benar dipakai

Sistem ini difokuskan ke kebutuhan operasional mini ISP: data pelanggan, pembayaran, keluhan, dan kontrol status layanan.

Billing Otomatis

  • Jatuh tempo mengikuti tanggal pasang
  • Reset status pembayaran tiap awal bulan
  • Auto isolir pelanggan yang lewat jatuh tempo

Pembayaran QRIS Midtrans

  • Generate invoice Snap dengan order ID berisi ID pelanggan
  • Webhook otomatis mengubah status menjadi lunas
  • Log pembayaran tersimpan ke sheet

Dashboard Pelanggan

  • Lihat tagihan, jatuh tempo, dan status sistem
  • Kirim keluhan dan lanjut chat WhatsApp
  • Akses broadcast info dari admin

Dashboard Admin

  • Statistik total pelanggan, aktif, dan terisolir
  • Kelola broadcast dan tiket keluhan
  • Monitoring daftar pelanggan lengkap

Akses ringan dari Android

Frontend dibuat tanpa framework agar cepat dimuat di browser Android dan mudah di-host di mana saja.

Mobile Friendly

Semua halaman responsif, ringan, dan bisa dipakai sebagai web app internal untuk admin maupun pelanggan.

Pembayaran Digital

Pelanggan bisa langsung generate pembayaran QRIS melalui Midtrans Snap dari dashboard client tanpa perlu kirim foto transfer manual.

Billing RTRW NET berbasis HTML, JavaScript, Google Apps Script, Google Sheets, Midtrans, dan simulasi Mikrotik.
